# 1️⃣ 개념 정리 

## 01. 통계로 요약하기

<평균 (Mean)
데이터 전체 값을 합한 뒤 데이터의 개수로 나눈 값으로, 데이터의 중심 위치를 나타낸다.
Series.mean(), numpy.mean()

중앙값 (Median)
데이터를 크기 순으로 정렬했을 때 가운데에 위치하는 값이다.
이상치의 영향을 적게 받기 때문에 대표값으로 자주 활용된다.
Series.median(), numpy.median()

분위수 (Quantile)
정렬된 데이터를 일정 비율로 나누는 기준값이다.
예를 들어 사분위수는 데이터를 25%, 50%, 75% 기준으로 나눈다.
Series.quantile()

분산 (Variance)
데이터가 평균을 기준으로 얼마나 퍼져 있는지를 나타내는 값이다.
값이 클수록 데이터의 변동성이 크다고 볼 수 있다.
numpy.var()

표준편차 (Standard Deviation)
분산의 제곱근으로, 원본 데이터와 동일한 단위를 사용한다.
따라서 분산보다 해석이 직관적이다.
numpy.std()

최빈값 (Mode)
데이터 중에서 가장 자주 나타나는 값이다.
특히 범주형 데이터 분석에서 많이 사용된다.
Series.mode()>

## 02. 분포 요약하기

<맷플롯립 (Matplotlib)
데이터 시각화를 위한 대표적인 라이브러리로, 다양한 그래프를 생성할 수 있다.
(산점도, 히스토그램, 박스플롯, 막대그래프, 선그래프 등)

산점도 (Scatter Plot)
두 변수의 값을 좌표평면 위에 점으로 표현하는 그래프이다.
변수 간 관계나 패턴을 파악할 때 유용하다.

히스토그램 (Histogram)
데이터를 일정 구간으로 나눈 후 각 구간에 속하는 데이터 개수를 시각화한 그래프이다.
데이터의 분포 형태를 확인할 수 있다.
데이터가 한쪽으로 치우친 경우에는 로그 스케일을 활용하면 표현이 더 효과적이다.

상자 수염 그림 (Box Plot)
데이터의 분포를 사분위수 기준으로 요약해서 보여주는 그래프이다.
Q1(제1사분위수)과 Q3(제3사분위수) 사이를 상자로 표현하고,
중앙값은 상자 내부 선으로 표시된다.
또한 IQR(사분위 범위)의 1.5배 범위 내의 값까지 수염으로 나타내어
데이터 분포와 이상치를 함께 확인할 수 있다.>
