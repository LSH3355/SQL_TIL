# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

<데이터 분석은 '유용한 정보를 발견하고 결론을 유추하거나, 의사 결정을 돕기 위해 데이터를 조사, 정제, 변환, 모델링하는 과정'으로 정의합니다. 데이터 분석은 다양한 접근 방법과 형태로 여러 비즈니스와 과학 분야에서 사용합니다. 현대 사회에서 데이터 분석은 비즈니스 결정을 과학적으로 내리기 위한 도구로 사용되는 경향이 있다.

데이터분석과 데이터 과학이 비슷하지만 다른 경향을 가지고 있다.
데이터 분석은 올바른 의사 결정을 돕기 위한 통찰을 제공하는 데 초점을 맞추고
데이터 과학은 한 걸음 더 나아가 문제 해결을 위한 최선의 솔루션을 만드는 데 집중>

## 02. 구글 코랩과 주피터 노트북

<구글 코랩은 구글이 대화식 프로그래밍 환경인 주피터 노트북을 커스터마이징한 것으로 웹 브라우저에서 파이썬 코드르 작성하고 실행할 수 있다. 따라서 컴퓨터에 파이썬 배포판을 설치하고 환경을 구축하는 데 시간을 쏟지 않고도 바로 실습을 진행할 수 있다.

위 내용에서 앞으로 협력을 통해 프로그래밍을 할 시에는 구글 코랩을 사용하는 것이 효율적일 것 같다는 생각을 하게 되었습니다.>

## 03. 이 도서가 얼마나 인기가 좋을까요?

<도서 데이터 찾기 해당 출판사 도서 데이터를 확인하려면 공개 데이터 세트를 찾아보거나 데이터 과학과 관련된 온라인 포럼에 질문을 올려서 도움을 요청할 수도 있습니다. 이 부분에서는 국내 대표적인 데이터 포털인 공공데이터포털에서 도서 관련 데이터가 있는지 찾아보는 부분이다.>

# 3️⃣ 확인 문제

## 문제 1.

> **🧚Q. 데이터 분석의 개념을 넓은 의미와 좁은 의미로 나누어 설명하고, 두 개념의 차이점도 함께 정리해주세요.**


```
1. 데이터 분석의 넓은 의미 

넓은 의미에서의 데이터 분석은 데이터를 수집하고 정리하며, 해석하여 의미 있는 정보를 도출하고 해석하여 의미 있는 정보를 도출하고 의사결정에 활용하는 전체 과정을 의미한다. 즉 단순히 숫자를 계산하는 것뿐만 아니라 문제 정의, 데이터 수집, 전처리, 분석, 결과 해석 까지 포함하는 포괄적인 활동이다.

2. 데이터 분석의 좁은 의미

좁은 의미에서의 데이터 분석은 이미 수집된 데이터를 통계적 방법이나 분석 기법을 이용하여 패턴이나 의미를 찾는 과정을 의미한다.

3. 두 개념의 차이점

1) 넓은 의미의 데이터 분석

의사결정과 문제 해결을 목적으로 포괄적이고 전략적인 개념

2) 좁은 의미의 데이터 분석

데이터 속 의미나 패턴 발견을 바탕으로 기술적으로 방법 중심의 개념

```
